# LexDraft Suite - Pricing Page

A comprehensive pricing page combining the best elements from three different AI-generated implementations (Gemini Pro, GPT-5.2-high, and Claude Opus 4.5).

## Features

### From Implementation #1 (Gemini Pro)
- **Framer Motion Animations**: Smooth transitions between pricing tabs with fade-in/out effects
- **Sticky Navbar**: Professional navigation that stays visible while scrolling
- **Gradient Backgrounds**: Beautiful visual design with radial gradients
- **Bundles Section**: Popular package combinations with clear pricing
- **FAQ Accordion**: Expandable questions with animated answers

### From Implementation #2 (GPT-5.2-high)
- **Mobile-Friendly Tables**: Comparison tables that transform into accordions on mobile
- **Detailed FAQ**: Comprehensive answers to common questions
- **Automation Use Cases**: Grid of what can be automated
- **"Best For" Labels**: Clear guidance on which plan suits different needs
- **Trust Bullets**: Quick value propositions in hero section

### From Implementation #3 (Claude Opus 4.5)
- **Currency Toggle**: Switch between EUR and USD pricing
- **Suite Pills**: Overview of all products in the ecosystem
- **Callout Panels**: Detailed explanations of AI billing and features
- **Better Data Structure**: TypeScript types and useMemo for performance
- **Decorative Backgrounds**: Subtle gradient orbs for visual depth
- **Segmented Tabs**: Clean tab navigation with descriptions
- **Professional Styling**: Glass morphism effects and modern design system

## Sections Included

1. **Hero Section**: Value proposition with trust badges
2. **Pricing Navigation**: Platform/AI/Automation tabs with billing and currency toggles
3. **Platform Pricing**: 4 tiers (Community, Starter, Team, Organization)
4. **AI Pricing**: General AI (3 tiers) + Legal AI (3 tiers)
5. **Automation Pricing**: 3 tiers (Basic, Plus, Autopilot)
6. **Bundles**: Popular combinations with assumptions
7. **FAQ**: 7 common questions with detailed answers
8. **Final CTA**: Call-to-action with feature summary

## Technologies Used

- **React 19**: Latest React with TypeScript
- **Tailwind CSS v4**: Modern utility-first CSS framework
- **Framer Motion**: Smooth animations
- **Lucide React**: Beautiful icons
- **Vite**: Fast build tool

```bash
npm run dev

# Build for production
npm run build

```

## Key Pricing Models

- **Self-hosting**: Free forever, unlimited users
- **Managed Platform**: €15-€99/workspace/month
- **General AI**: €0 (BYO) or €10-€25/user/month
- **Legal AI**: €0 (BYO) or €99-€199/workspace/month
- **Automation**: €0 (Basic), €29 (Plus), €79 (Autopilot)/workspace/month
- **Yearly Billing**: 20% discount

## Design Principles

- **Trust-first**: Self-hosting always remains an option
- **Predictable**: No per-token billing, flat rates with soft limits
- **Flexible**: Bring your own AI keys or use managed AI
- **Clear**: Detailed comparisons and transparent assumptions
